# Neibor

## Description

Neighbourhood social alert application.
Uses a monorepo structure
Uses microservice architecture

Frontend: React
Backend: Node

## Install

Clone repository and install on local machine

To install:
`npm install`

Stall packages into workspaces:
`npm i PACKAGE -w=WORKSPACE`

for example

`npm i lodash -w=web`

## Start project

To start project:

`npm start`

To start backend:

`npm start -w=backend`

To start frontend:

`npm run dev -w=web`
